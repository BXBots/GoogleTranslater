# GoogleTranslater

A simple Google Tranlator Bot. Languages :

#English, #Tamil, #Telugu, #Hindi, #Kannada, #Malayalam, #Urdu, #Punjabi, #Spanish, #Korean, #Japanese, #Chinese, #Greek, #Italian, #Nepali

## Bot Command's

/start - Start
/help - Available Languages
